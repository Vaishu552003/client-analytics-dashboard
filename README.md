# 🚀 Client Analytics & Decision Dashboard

A full-stack analytics dashboard built using **Flask, Pandas, Chart.js, and Bootstrap**.

This project allows users to upload CSV or Excel files and generate interactive business insights including revenue, profit, region performance, and product profitability.

---

## 📊 Features

- 📁 Upload CSV / Excel files
- 📈 Monthly Revenue Trend (Line Chart)
- 🌍 Revenue by Region (Bar Chart)
- 📦 Profit by Product (Bar Chart)
- 📌 Key Business KPIs:
  - Total Revenue
  - Total Profit
  - Total Units Sold
  - Deal Close Rate
- 🌙 Professional Dark UI
- ⚡ Interactive Charts using Chart.js

---

## 🛠️ Tech Stack

**Backend**
- Python

- Pandas

**Frontend**
- HTML
- Bootstrap 5
- Chart.js

**Database**
